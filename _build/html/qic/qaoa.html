Quantum Approximate Optimization Algorithm (QAOA)#

Definition 1

Consider the combinatorial problem Max E3LIN2 over n bits. The E3 means that each clause contains exactly 3 variables. The LIN2 means that each constraint is a linear equation mod 2 so for say bits \(x_1\), \(x_2\) and \(x_3\) the constraint is either \(x_1+x_2+x_3 = 0\) or \(x_1+x_2+x_3 = 1\).
